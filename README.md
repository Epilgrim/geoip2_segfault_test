# App to reproduce Geoip segfault

## Steps to reproduce

1. Clone this repository

    ```
    git clone git@github.com:Epilgrim/geoip2_segfault_test.git
    ```

1. Install dependencies

    ```
    cd geoip2_segfault_test
    composer install
    ```

1. Execute tests

    ```
    bin/phpspec run
    ```
